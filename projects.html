Hostel Reservation System

Description: A web-based application designed to manage hostel room bookings for students.

This system includes features for students to register, browse available hostel rooms, make reservations, and view booking history. Admins can manage room availability and student records. It showcases full-stack development capabilities, focusing on database integration, user authentication, and CRUD operations.

Fun Typing Speed Checker

Description: An interactive tool that measures typing speed and accuracy in real-time.

The application presents random text passages to the user and tracks the number of correct words typed in a given time limit. It helps users improve typing skills by providing instant feedback. This project emphasizes JavaScript programming, event handling, and user experience design.
